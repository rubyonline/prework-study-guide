Prework Study Guide

HTML

  • The doctype element lets the browser know you will be working in HTML.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta tag defines metadata in the head.
  • The base tag defines the base URL for the website
  • The link tag connects an external link
  • The style tag allows us to use and define CSS Styles within the webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Use a period before a class
  • The border proprety assigns a boarder
  • You can use the box-shadow proprety to create a shadow

Git

  • Use "git status" to check which branch you are in.
  • Use "git checkout -b "yourbranchname"" to make a new branch.
  • Use "git add -A" to add all your changes to your document.
  • Use "git commit -m "yourcommenthere"" to commit your new changes.
  • Use "git pull origin main" to pull the work from main.
  • Use "git push origin [your branch name here]" to push changes to the branch.
